🛒 Vijay Shop – Spring Boot E-Commerce

A full-stack E-Commerce web application developed using Spring Boot, Spring Security, Hibernate, JSP, and H2 Database. The application provides role-based authentication for Admin and Users, allowing secure product management, shopping cart functionality, and customer management through a structured MVC architecture.

---

📖 Project Overview

Vijay Shop is a role-based E-Commerce web application that enables users to browse products, manage carts, and maintain profiles, while administrators can manage products, categories, and customers through a dedicated dashboard.

---

🚀 Features

- 🔐 Role-Based Authentication & Authorization using Spring Security
- 🔒 BCrypt Password Encryption
- 🛍️ Product Browsing & Management
- 🛒 Shopping Cart Functionality
- 👤 User Registration & Profile Management
- 📂 Category Management
- 👥 Customer Management
- ⚙️ Admin Dashboard
- 🎨 Responsive UI using Bootstrap

▶️ How to Run

Clone Repository

git clone https://github.com/vijayakumardevforge/vijay-shop-springboot-ecommerce-.git

Navigate to Project

cd vijay-shop-springboot-ecommerce

Run Application

./mvnw spring-boot:run

Open Browser:

http://localhost:8080
